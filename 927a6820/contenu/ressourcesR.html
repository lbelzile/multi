Ressources R

Il existe plusieurs ressources en ligne pour apprendre R et le tidyverse, notamment

  • Les ressources didactiques listées sur le CRAN
  • Introduction to R and RStudio par Open Intro Stat
  • Teacups, giraffes & statistics: révise les concepts statistiques et les bases de la programmation
  • le livret RYouWithMe de R-Ladies Sydney
  • le livre R for Data Science, qui adhère aux principes du tidyverse.
  • le paquet R DoSStoolkit, développé à l’université de Toronto
  • la documentation d’introverse
  • les feuilles d’aide mémoire de RStudio, aussi disponibles depuis le menu Aide dans RStudio

Vous pouvez installer le paquet hecmulti, de même que les paquets employés dans les notes de cours,à l’aide des commandes suivantes:

lib <- c("remotes", "psych", "glmnet", "caret", 
         "leaps", "car", "MASS", "glmbb", "AER",
         "tidyverse", "nnet", "mice", "survminer", 
         "coin","survival", "patchwork", "sampleSelection",
         "Hmisc", "aod", "cluster", "NbClust",
         "emmeans", "ROCR", "factoextra", "muhaz",
         "summarytools", "flexclust", "dummies", "gridExtra"
         )
for(pack in lib){
  if(!require(pack, quietly = TRUE)){
    install.packages(pack)
  }
}

remotes::install_github("lbelzile/hecmulti")
